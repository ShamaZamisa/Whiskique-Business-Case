# Whiskique Business Case Study
# Project Overview

This project analyzes sales, profitability, shipping costs, and customer behavior for a pet retail business using Power BI.

The goal was to:

- Identify revenue and margin drivers

- Optimize shipping costs

- Analyze customer lifetime value

- Perform market basket analysis

- Provide actionable business recommendations

Key Metrics: 
  - Total Sales: $1.55M
  - Total Profit: $432K
  - Profit Margin: 27.82%
  - Baseline Shipping Cost: $385K
